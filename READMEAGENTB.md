# BP Fundamental Analysis Project

## Project Overview
This project conducts a fundamental equity analysis of BP plc using financial data
from 2020 to 2024. The analysis integrates data collection, cleaning, financial ratio
analysis, and valuation models to evaluate the company’s performance and intrinsic value.

## Data Sources
- Yahoo Finance
- Alpha Vantage
- BP Annual Reports

## Main Tasks
- Financial statement collection (Income Statement, Balance Sheet, Cash Flow)
- Data cleaning and validation
- Financial ratio analysis
- Free Cash Flow (FCF) calculation
- Discounted Cash Flow (DCF) valuation
- Dividend Discount Model (DDM) valuation
- Relative valuation using market multiples
